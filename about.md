# About This Site

This is an example page written entirely in **Markdown**!

## How Jekyll Works

1. You write pages in Markdown (`.md` files)
2. Add "front matter" at the top (the stuff between `---`)
3. Push to GitHub
4. GitHub automatically converts it to HTML

## Example Markdown Features

### Code Blocks

```python
def hello_world():
    print("Hello from app.kylepace.net!")
```

### Lists

- Easy to write
- No HTML tags needed
- Just use Markdown syntax

### Links

Check out my [main blog](https://kylepace.net) or my [GitHub profile](https://github.com/KylePace1).

### Emphasis

You can use **bold**, *italic*, or ***both***!

## Creating New Pages

Just create a new `.md` file with front matter:

```markdown
---
layout: default
title: My New Page
---

# My New Page

Content goes here...
```

Then commit and push. That's it!
